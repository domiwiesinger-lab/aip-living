# AIP Living · Mountain View Estates

Landing-Page für voll erschlossene See- und Berggrundstücke am Khao Sam Roi Yot Nationalpark, Pranburi (Thailand).

**Live-Demo:** siehe GitHub Pages Link in den Repo-Einstellungen.

## Features
- Statisches Single-File HTML (kein Build-Step)
- Drei Sprachen: Deutsch, English, ภาษาไทย — Live-Umschaltung via Buttons
- Video-Hero, Parallax, Scroll-Reveal-Animationen
- Sechs Bauformen (A-Frame, Pool Villa, Bungalow, Loft, Sala, Custom)
- Lead-Formular mit Plot- / Bauform- / Zeitfenster-Auswahl
- Standort-Karte mit Distanzen zu Strand, Nationalpark, Höhle, Klinik, Airport, Bangkok
- Vollständig responsive (Mobile-First Breakpoints bei 1080 / 960 / 680 / 600px)

## Stack
HTML5 + CSS3 + Vanilla JavaScript · keine Frameworks · keine Build-Tools.

## Lokal öffnen
`index.html` einfach im Browser öffnen.
